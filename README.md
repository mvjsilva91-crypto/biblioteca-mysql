# 📚 Banco de Dados - Biblioteca

Projeto de banco de dados relacional desenvolvido para simular o funcionamento de uma biblioteca.

---

## 📌 Funcionalidades

* Cadastro de autores e livros
* Controle de membros
* Registro de empréstimos
* Consultas SQL com JOIN e agregações

---

## 🗂 Estrutura do Projeto

```
biblioteca-database-mysql
 ├── sql/
 │   ├── 01_create_database.sql
 │   ├── 02_create_tables.sql
 │   ├── 03_inserts.sql
 │   └── 04_queries.sql
 └── README.md
```

---

## 🧠 Modelagem do Banco

* **Autor**: escritores dos livros
* **Livro**: informações dos livros
* **Membro**: usuários da biblioteca
* **Emprestimo**: controle de empréstimos

---

## ⚙️ Tecnologias

* MySQL

---

## ▶️ Como executar

1. Executar `01_create_database.sql`
2. Executar `02_create_tables.sql`
3. Executar `03_inserts.sql`
4. Executar `04_queries.sql`
